# 🛡️ Cyber Recon Toolkit

A set of reconnaissance (Recon) and Open Source Intelligence (OSINT) tools that I researched and utilized for learning and practicing penetration testing on Kali Linux.

This GitHub repo is used by me as my personal notes about commonly used reconnaissance tools: their purpose, how to install, usage example and screenshots of practical usage.

> **Disclaimer:** This repo is made purely for educational and ethical security testing purposes. Obtain proper authorization before doing any reconnaissance or security assessment.
